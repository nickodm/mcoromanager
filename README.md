# MCoroManager

A simple program to handle directories created by students to do the Moisés Coronado's tasks.

# Usage

### To create a new folder
```bash
mcoromanager new
```

### Done

When you just finished with all of your tasks and you are ready to send your MD5 to the teacher, 
you can use this command to:

* Verify your folder's name.
* Zip your folder to **.7z**.
* Calculate the folder's **MD5** and save it in a **.txt** file.

```bash
mcoromanager done
```
